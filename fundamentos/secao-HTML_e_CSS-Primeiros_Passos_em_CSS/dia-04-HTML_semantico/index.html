Fatos sobre o Stomatopoda

Primeiro contato imediato com o Stomatopoda

Os Stomatopoda ou (Odontodactylus scyllarus) são conhecidos por diversos nomes, neste artigo vamos chama-los de camarão boxeador, pois estes animais são excelentes golpeadores, eles "socam" suas presas com suas garras com uma força de até 2,5 mil vezes seu peso em menos de 800 microsegundos. Este nocaute equivale a um tiro de pistola calibre 22.
